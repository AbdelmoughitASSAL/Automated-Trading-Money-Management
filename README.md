# Automated Trading - Money Management Case

* **Description**

In this project, we developed an automated trading system focused on *Money Management*.

Chapter 3, which you will find in this repository, presents a practical case of risk management and optimal capital allocation in a trading system.

The main objective of this system is to determine the optimal fraction of capital to invest in each trade based on risk management parameters such as position size, risk level, and market volatility.

**Data Download from MetaTrader 5 (MT5)**

Historical data will be automatically downloaded from MetaTrader 5 (MT5) for two trading symbols: **EURUSD** and **US500Cash**.

* **EURUSD**: Historical data for EUR/USD will be downloaded and saved in the file `data_analyzed1.csv`.
* **US500Cash**: Historical data for the S&P 500 index (US500Cash) will be downloaded and saved in the file `data_analyzed2.csv`.

These CSV files will be used for analysis and modeling in this project.

**Prerequisites:**

* MetaTrader 5 (MT5) installed and configured.
* Python libraries: `MetaTrader5`, `pandas`, `matplotlib`, `datetime`, `scikit-learn`.
